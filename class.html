🧱 HTML Basics – Structure & Syntax

1. 💡 What is HTML?

HTML (HyperText Markup Language) is the standard language used to create webpages. It uses tags to structure content.

2. 🏗️ Basic HTML Document Structure

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

Explanation:

TagPurpose
<!DOCTYPE html>Declares the document as HTML5
<html>Root of the HTML page
<head>Contains metadata (not shown on page)
<title>Sets the title (shown in browser tab)
<body>Visible content of the page

3. 🏷️ HTML Tags

Tags are enclosed in angle brackets: <tagname>

Most tags have an opening and closing pair:

<p>This is text</p>

Some tags are self-closing:

<img src="image.jpg" alt="description">
<br>
<hr>

4. 🔤 Common HTML Tags

TagUse
<h1> to <h6>Headings, from largest to smallest
<p>Paragraph
<a href="url">Link
<img src="..." alt="...">Image
<ul>, <ol>, <li>Lists
<div>Generic block container
<span>Generic inline container
<strong>, <em>Bold and italic text

5. 🎨 HTML Attributes

Attributes give more information about tags. They are written in the opening tag.

<a href="https://example.com" target="_blank">Visit Example</a>
AttributeMeaning
hrefLink URL (for <a>)
srcImage or script source
altAlternate text for images
idUnique identifier
classReusable styling hook
styleInline CSS
targetWhere to open a link (_blank, _self)

6. 🧩 Nesting Elements

HTML elements can be nested inside each other. Always properly close tags!

✅ Good:

<p>This is a <strong>bold</strong> word.</p>

❌ Bad:

<p>This is a <strong>bold</p></strong>

7. 💬 Comments

Use comments to leave notes in your code:

<!-- This is a comment -->

8. ⚠️ Best Practices

  • Always close your tags.
  • Use lowercase for tags and attributes.
  • Keep nesting clear and logical.
  • Use semantic tags like <header>, <footer>, and <main> for better accessibility and SEO.

üí° What is HTML?

HTML (HyperText Markup Language) is the standard language used to create webpages. It uses tags to structure content.


üèóÔ∏è Basic HTML Document Structure 

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

Explanation:

TagPurpose
<!DOCTYPE html>Declares the document as HTML5
<html>Root of the HTML page
<head>Contains metadata (not shown on page)
<title>Sets the title (shown in browser tab)
<body>Visible content of the page

üè∑Ô∏è HTML Tags

Tags are enclosed in angle brackets: <tagname>

Most tags have an opening and closing pair:

<p>This is text</p>

Some tags are self-closing:

<img src="image.jpg" alt="description">
<br>
<hr>
<br>: Line break. Used to start a new line without a new paragraph.
Example: <p>Hello<br>World</p>
Output:

Hello
World

<hr>: Horizontal rule. Used to create a thematic break in the page.
Example: <hr>

üìã Tables

Below is the HTML to create a table . Notice the tags! 
<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>24</td>
  </tr>
</table>
NameAge
Alice24
<tr>: Table row. Used inside <table> to define a row.
Example: <tr><td>Cell</td></tr>
Output:
Cell
<td> and <th>: Table cell and table header cell respectively.
Example: <th>Name</th> <td>John</td>
Output:
NameJohn
